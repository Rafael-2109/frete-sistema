Como Funciona
Para que serve?

O De-Para traduz nossos códigos internos para os códigos que o portal do Atacadão reconhece, permitindo agendamentos automáticos.

Quando atualizar?
  • Ao cadastrar novos produtos
  • Quando o Atacadão alterar códigos
  • Para corrigir erros de mapeamento
Fator de Conversão

Use quando a unidade de medida for diferente. Ex: Se vendemos em caixas mas o Atacadão compra em unidades, use fator 12 (1 caixa = 12 unidades).

Formato de Planilha para Importação

Para importar via arquivo xlsx/csv, use o seguinte formato (ou baixe o modelo acima):

cod atacadao * descricao atacadao nosso cod * descricao nosso fator conversao cnpj cliente observacoes
82545 AZEITONA VERDE CAMPO BELO FAT.POUCH 30x80G 4310146 auto 1.0
46624 AZEITONA VERDE CAMPO BELO POUCH 18x180G 4310152 auto 1.0
Ao reimportar: Se o mapeamento já existir, a descrição do Atacadão será atualizada automaticamente.
Dica: Use "Exportar Planilha" para baixar os mapeamentos atuais, edite e reimporte.
