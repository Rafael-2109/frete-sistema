De-Para EAN (Produtos)

O De-Para EAN é essencial para o funcionamento dos agendamentos no Portal Tenda. Ele converte nossos códigos internos de produtos para os códigos EAN (código de barras) que o portal reconhece.

Importante: Sem o De-Para configurado, o sistema não conseguirá criar agendamentos para os produtos.

Como Usar
Para que serve?

O portal Tenda identifica produtos pelo código EAN (código de barras). Este De-Para conecta nossos códigos internos aos EANs do Tenda.

Quando atualizar?
  • Ao cadastrar novos produtos
  • Quando o Tenda alterar códigos
  • Para corrigir erros de mapeamento
Fator de Conversão

Use quando a unidade de medida for diferente. Ex: Se vendemos em caixas mas o Tenda compra em unidades, use fator 12 (1 caixa = 12 unidades).

Formato XLSX para Importação

Para importar via planilha, use o seguinte formato:

codigo_nosso descricao_nosso ean descricao_ean fator_conversao
PROD001 Produto Exemplo 7891234567890 Produto no Portal 1.0
PROD002 Outro Produto 7899876543210 Descrição Portal 12.0
Dica: Você pode obter os códigos EAN diretamente com o Tenda ou consultando as notas fiscais de compra.
