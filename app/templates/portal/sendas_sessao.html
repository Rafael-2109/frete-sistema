📋 Como Capturar os Cookies
⚠️ Atenção: Cookies de Terceiros
Se você vir avisos sobre "Third-party cookie will be blocked":
• Vá em Chrome → Configurações → Privacidade → Cookies
• Escolha "Permitir todos os cookies" temporariamente
• Ou adicione [*.]trizy.com.br como exceção
  1. Abra o Chrome (não use modo anônimo)
  2. Acesse: https://login.trizy.com.br/access/auth/login/
  3. Faça login normalmente (resolva o captcha manualmente)
  4. Após logar, pressione F12 para abrir o DevTools
  5. Vá para aba Console
  6. Cole e execute o código abaixo:
// Script melhorado para capturar TODOS os dados de autenticação const captureData = { // Cookies em formato array com atributos cookies: document.cookie.split('; ').map(c => { const [name, ...value] = c.split('='); return { name: name, value: value.join('='), domain: '.trizy.com.br', // Importante para subdomínios path: '/', secure: true, httpOnly: false, sameSite: 'None' // Permite cross-site }; }), // Cookies raw para backup cookiesRaw: document.cookie, // Todos os storages localStorage: {}, sessionStorage: {}, tokens: {}, // Informações do contexto domain: window.location.hostname, url: window.location.href, timestamp: new Date().toISOString() }; // Capturar localStorage completo Object.keys(localStorage).forEach(key => { captureData.localStorage[key] = localStorage.getItem(key); if (key.toLowerCase().includes('token') || key.toLowerCase().includes('auth') || key.toLowerCase().includes('bearer') || key.toLowerCase().includes('jwt')) { captureData.tokens[key] = localStorage.getItem(key); } }); // Capturar sessionStorage completo Object.keys(sessionStorage).forEach(key => { captureData.sessionStorage[key] = sessionStorage.getItem(key); if (key.toLowerCase().includes('token') || key.toLowerCase().includes('auth') || key.toLowerCase().includes('bearer') || key.toLowerCase().includes('jwt')) { captureData.tokens[key] = sessionStorage.getItem(key); } }); // Copiar para área de transferência copy(JSON.stringify(captureData)); // Logs detalhados console.log('✅ Dados capturados e copiados!'); console.log('📊 Resumo:'); console.log(' • Cookies:', captureData.cookies.length, 'items'); console.log(' • LocalStorage:', Object.keys(captureData.localStorage).length, 'items'); console.log(' • SessionStorage:', Object.keys(captureData.sessionStorage).length, 'items'); console.log(' • Tokens:', Object.keys(captureData.tokens).length, 'items'); console.log(' • URL:', captureData.url); console.log('\n🔴 IMPORTANTE: Cole (Ctrl+V) no formulário abaixo!');
