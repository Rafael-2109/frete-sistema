Tipos de Custo
Médio Mês Média ponderada dos custos de compra no período (mês). Fórmula: (Valor Total Líquido) / (Qtd Comprada)
Último Custo Custo unitário da última compra realizada no período, descontando impostos proporcionalmente.
Médio Estoque Custo médio ponderado considerando estoque inicial + compras. Fórmula: (Custo Inicial + Custo Compras) / (Qtd Inicial + Qtd Comprada)
Custo Considerado Custo definido para uso nos pedidos. COMPRADOS: editável. PRODUZIDOS: calculado automaticamente via BOM.
Dica: Clique nos valores de produtos produzidos para expandir a lista de materiais (BOM).
