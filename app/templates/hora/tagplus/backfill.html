Importa NFs já emitidas no TagPlus para o sistema HORA (modo UPSERT).
  • O sistema chama GET /nfes filtrando pela data de emissão e itera todas as NFs do período (todos os status).
  • Para cada NF, chama GET /nfes/{id} e cria/atualiza HoraVenda + itens diretamente do JSON estruturado da API — sem PDF, sem LLM.
  • Resolução do modelo via tagplus_codigo (cadastrado em /hora/modelos).
  • UPSERT: NF já existente recebe atualização de campos vazios (endereço, telefone, etc). Edições manuais do operador são preservadas.
  • Status TagPlus tratados: A=Aprovada → cria/atualiza · S=Cancelada / 4=Inutilizada → cancela venda + libera chassis · 2=Denegada / N=Em digitação → pula.
  • Divergências são registradas mas não bloqueiam (CHASSI_NAO_CADASTRADO, LOJA_DIVERGENTE, CNPJ_DESCONHECIDO, TABELA_PRECO_AUSENTE).
