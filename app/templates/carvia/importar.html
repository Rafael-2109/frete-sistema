Como Funciona
  1. Upload: Selecione NFs (PDF ou XML), CTes (XML) e/ou Faturas (PDF) simultaneamente.
  2. Parsing: O sistema extrai dados automaticamente de cada arquivo. PDFs de fatura com multiplas paginas geram uma fatura por pagina.
  3. Matching: CTes sao associados as NFs pela chave de acesso ou CNPJ+numero.
  4. Review: Voce confere e corrige os dados antes de confirmar.
  5. Salvar: NFs, CTes (operacoes/subcontratos) e faturas sao criados.
Formatos Aceitos
  • PDF DANFE: NF de mercadoria (detectado pela chave de acesso)
  • PDF Fatura: Fatura de frete (sem chave de acesso no PDF)
  • XML NF-e: Nota Fiscal Eletronica
  • XML CTe: Conhecimento de Transporte
