# Opção 104 — Faturas Repassadas para Agências

> **Módulo**: Comercial/Financeiro
> **Referência interna**: Opção 466
> **Páginas de ajuda**: 1 página consolidada
> **Atualizado em**: 2026-02-15

## Função

Relaciona as faturas que foram repassadas para agências ou parceiros. Interface de consulta com filtros para localizar faturas específicas no contexto de repasse comercial.

## Quando Usar

- Consultar quais faturas foram repassadas para agências/parceiros
- Acompanhar repasses realizados pelas opções 104 e 457
- Filtrar faturas por data de emissão, vencimento ou sigla da agência

## Campos / Interface

### Filtros Disponíveis

- **Data de emissão**: Período de emissão das faturas
- **Data de vencimento**: Período de vencimento das faturas
- **Sigla da agência**: Identificação da agência/parceiro que recebeu o repasse

## Integração com Outras Opções

- **Opção 457**: Também utilizada para repasse de faturas para agências/parceiros
- Ambas as opções (104 e 457) alimentam o mesmo registro de repasses

## Observações e Gotchas

- A tela referencia internamente a opção 466 na ajuda, mas o acesso funcional é pela opção 104
- O SSW reutiliza a mesma página de ajuda entre diferentes pontos de entrada do menu
- A consulta permite combinação de múltiplos filtros para refinar resultados
